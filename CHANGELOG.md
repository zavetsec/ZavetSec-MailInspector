# Changelog

Все значимые изменения ZavetSec-MailInspector фиксируются здесь.
Формат — [Keep a Changelog](https://keepachangelog.com/); версионирование — [SemVer](https://semver.org/).

## [1.2] — 2026-06-12

### Добавлено
- **Рекурсивный анализ архивов** — вложенное содержимое архивов распаковывается
  и повторно сканируется полным набором детекторов (реальный тип, опасные/двойные
  расширения, exec-под-документ, энтропия, макросы, вложенная защита). Хэши
  вложенных файлов попадают в список IOC и показываются деревом путей в отчёте.
  - ZIP / TAR / GZIP распаковываются **в памяти** (stdlib); 7z / RAR — через
    опциональные `py7zr` / `rarfile`. OOXML-документы не считаются контейнерами.
- **Защита от zip-bomb / ресурсов** — лимиты по глубине (3), числу файлов (500),
  размеру на файл (64 МБ) и суммарному бюджету (256 МБ), плюс проверка
  compression-ratio. Подозрение на decompression bomb фиксируется, распаковка
  останавливается, а не «детонирует». Обработка в памяти снимает path-traversal (zip-slip).
- Таблица вложений в отчёте показывает вложенные файлы деревом с отступами.

### Опциональные зависимости
- `py7zr` — рекурсия в 7-Zip архивы.
- `rarfile` — рекурсия в RAR (и улучшенный детект шифрования RAR).

## [1.1] — 2026-06-12

### Добавлено
- **Контекстная энтропия вложений** (Shannon, бит/байт). Высокая энтропия
  флагается только там, где она *неожиданна*: packed/obfuscated исполняемые
  файлы и «документы», которые на деле — непрозрачные blob'ы. Изначально
  высокоэнтропийные форматы (ZIP/OOXML/PNG/JPEG/PDF/архивы) не дают ложных.
- **Детект защищённых паролем архивов** для ZIP (включая WinZip-AES), RAR4,
  RAR5 (шифрование заголовков) и 7-Zip (AES-coder), с мягкой best-effort
  обработкой и нулём ложных на OOXML.
- **Корреляция пароля из тела** — зашифрованный архив, пароль к которому
  присутствует в теле письма, эскалируется до HIGH (классический malspam
  с обходом AV/песочницы).
- Энтропия и защита выводятся в таблице HTML-отчёта и в JSON.

### Опциональная зависимость
- `rarfile` — улучшает покрытие RAR (детект RAR4 работает и без неё).

## [1.0] — 2026-06-12

### Добавлено
- Первый публичный релиз.
- Разбор `.eml` (RFC822) и `.msg` (Outlook).
- Анализ аутентификации: SPF / DKIM / DMARC, цепочка `Received`, originating IP.
- Спуфинг отправителя: несовпадение From/Return-Path/Reply-To, чужой адрес
  в display-name, имитация бренда, несовпадение домена Message-ID.
- Анализ URL: link spoofing, IP-literal хосты, обфускация `user:pass@`,
  punycode/IDN и homoglyph со смешением алфавитов, сокращатели, подозрительные TLD,
  бренд-в-поддомене, опасные схемы URI.
- Анализ тела: двуязычный (RU/EN) скоринг социальной инженерии, tracking-пиксели,
  скрытый текст, HTML-формы в письме.
- Анализ вложений: MD5/SHA-1/SHA-256, реальный тип по magic bytes vs расширение,
  опасные и двойные расширения, детект VBA-макросов (oletools), пометка архивов.
- Опциональный онлайн threat-intel (MalwareBazaar + ThreatFox) за флагом `--online`.
- Самодостаточный HTML-отчёт (ноль внешних ссылок), вывод в JSON, цветная консоль.
- Риск-скоринг с убывающей отдачей по категориям; пороги вердиктов.
- Рекурсия по каталогам; коды возврата отражают худший вердикт.
