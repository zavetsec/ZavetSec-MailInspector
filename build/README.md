# Сборка автономных артефактов

Базовый способ раздачи — сам скрипт `.py`: его видно глазами, и он работает везде
(для `.eml` нужна только стандартная библиотека). Скрипты ниже собирают портативные
артефакты для тех, у кого Python не стоит.

## `.pyz` — портативный, к AV дружелюбный (рекомендуется)

Нативный Python zipapp — один файл, компактный, антивирус его не флагает.

```bash
build/build.sh pyz          # Linux/macOS
build\build.ps1 -Target pyz # Windows
# запуск: python ZavetSec-MailInspector.pyz message.eml -o report.html
```

На целевом хосте нужен Python. С `oletools` файл весит ~30 МБ; если оставить только
stdlib, будет заметно легче.

## `.exe` — Python не нужен (для внутреннего использования)

```bash
build/build.sh exe          # Linux (кросс-сборка не поддерживается; для Windows собирайте на Windows)
build\build.ps1             # Windows -> dist\ZavetSec-MailInspector.exe
```

> Неподписанный однофайловый PyInstaller‑бинарь в рантайме распаковывает интерпретатор,
> и эвристики AV/EDR на это часто срабатывают (false positive). Поэтому правильнее не
> раздавать бинарь публично, а добавить его SHA‑256 в allowlist эндпоинт‑политики.
> `build.ps1` печатает SHA‑256 после успешной сборки.

`--collect-submodules` для `oletools` и `extract_msg` обязателен: оба подгружают модули
динамически, и без него onefile‑сборка падает в рантайме.
